# PRODUCT_SPEC.md — 제품 명세 SSOT

## 1. 앱 개요

| 항목 | 내용 |
|------|------|
| 앱 이름 | 임장노트 |
| 목적 | 부동산 임장(현장 방문) 시 매물 정보를 기록하고 비교하는 앱 |
| 타겟 사용자 | 집을 구하는 개인 사용자 |

---

## 2. 핵심 기능

### 무료 플랜
- 매물 등록 (주소, 가격, 층수, 평수, 방문 날짜)
- 사진 첨부 (매물당 최대 5장, Supabase Storage)
- 간단 메모
- 매물 최대 **3개** 저장

### 프리미엄 플랜 (월 4,900원) — v2 이후
- 매물 무제한 저장
- 사진 무제한
- 매물 비교표 자동 생성
- 체크리스트 (채광, 소음, 주차, 수압 등)
- 지도에 매물 위치 표시 (Kakao Maps API)
- PDF 내보내기
- 가족/배우자와 공유

---

## 3. MVP 범위 (실습 대상)

> 결제·지도·PDF·공유는 v2 이후. 이번 실습은 아래 범위만 구현한다.

### MVP 기능
1. **인증**: Supabase Auth (이메일/패스워드 로그인·회원가입)
2. **매물 CRUD**: 등록, 목록, 상세, 수정, 삭제
3. **사진 업로드**: Supabase Storage (매물당 최대 5장)
4. **체크리스트**: 채광·소음·수압·주차·엘리베이터 별점(1~5) 기록
5. **비교표**: 선택한 매물 2~3개 나란히 비교
6. **무료 한도**: 비로그인 차단, 무료 계정은 매물 3개 초과 시 안내

## 4. 매물 기록 항목

### 기본 정보
| 필드 | 타입 | 설명 |
|------|------|------|
| address | text | 주소 (필수) |
| deposit | integer | 보증금 (만원) |
| monthly_rent | integer | 월세 (만원) |
| sale_price | integer | 매매가 (만원) |
| area_sqm | numeric | 평수 |
| floor | integer | 층수 |
| room_count | integer | 방 개수 |
| visit_date | date | 방문 날짜 |
| agent_contact | text | 부동산 연락처 |
| memo | text | 자유 메모 |

### 체크리스트 (별점 1~5)
| 항목 | 필드명 |
|------|--------|
| 채광 | sunlight |
| 소음 | noise |
| 수압 | water_pressure |
| 주차 가능 여부 | has_parking |
| 엘리베이터 유무 | has_elevator |
| 관리비 (만원) | management_fee |

## 5. 화면 목록

| 화면 | 경로 | 설명 |
|------|------|------|
| 로그인 | `/login` | 이메일 로그인 |
| 회원가입 | `/signup` | 신규 가입 |
| 대시보드 | `/dashboard` | 매물 목록 |
| 매물 등록 | `/dashboard/new` | 신규 매물 작성 |
| 매물 상세 | `/dashboard/[id]` | 상세 보기 + 사진 |
| 매물 수정 | `/dashboard/[id]/edit` | 수정 폼 |
| 비교표 | `/dashboard/compare` | 2~3개 매물 비교 |
